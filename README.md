## Setup

```
$ gem install jekyll
$ jekyll new robc.jekyll.blog
$ cd robc.jekyll.blog
$ jekyll serve
# => Now browse to http://localhost:4000

# Plugin
$ gem install jekyll-textile-converter
$ gem install jekyll-toc jekyll-tagging jekyll_figure
```

## Config

```
gems:
- jekyll-textile-converter

textile_ext: "textile,txtl,tl"
```
